Steps to execute the project:-
1. Download and install xampp.
2. Extract the zip file and paste at the following path:-C:\xampp\htdocs.
3. Start Xampp and start apache and sql server.
4. click on admin button in front of sql server in xampp control panel to start phpmyadmin.
5. import the "housing.sql" into the phpmyadmin.
6. start your Web Browser and open the project via localhost. type:- "localhost/Housing_Management/Home.php".
7. The Project will start.
8. Chech the password for the admin account from the php my admin.
9. Login and operate.
